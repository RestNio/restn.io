About

Hello fellow cloudsters :-) Want to create a simple or very complicated web-API in NodeJS? RestNio aims to provide you with an intuative yet comprehensive framework to do so.
RestNio was originally started by Kasper as a small hobby project back in 2018 to learn NodeJS and webstandards, but it has never stopped gradually growing. Since then it has been used in many API applications; from normal websites to (IOT) admin panels, conferencing software and even online games.
RestNio is similar to frameworks such as Express.js or Snails.js. The framework is not necesarrily better, but offers some interesting differences. RestNio offers a custom routing system with a heavy focus on regular expressions. The other focus is a layer of abstraction in the routes allowing route definitations to be shared between http and websocket requests, with plans to support other protocols such as MQQT in the future.
RestNio is currently in Beta, there are no known issues yet but you should use it at your own risk. We'd love for more people to test this software and give feedback.
